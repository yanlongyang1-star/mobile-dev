# Firebase Test Lab Script

## Purpose

Use this script to capture Assessment 4 evidence for Firebase Test Lab. Each student should run a different device configuration and include screenshots of the matrix, logs and result.

## Build APK

```bash
npm run typecheck
npm run lint
npm run test
npm run build:android:preview
```

Download the APK from the EAS build page.

## Test Lab Run

1. Open Firebase Console.
2. Go to Test Lab.
3. Upload the preview APK.
4. Choose Robo test.
5. Select a device.
6. Start the test.
7. Capture screenshots of:
   - device model and Android version
   - test matrix status
   - logs
   - screenshots/video generated by Test Lab

## Suggested Student Split

| Student | Device | Focus |
| --- | --- | --- |
| Student A | Pixel 7, Android 14 | Login, Browse, Booking |
| Student B | Pixel 5, Android 13 | Campus GPS, Battery, Notification |

## Notes for Report

Mention that native features such as AdMob, background location and notifications require a development or preview build rather than Expo Go. This is expected for Expo apps that include custom native modules.
